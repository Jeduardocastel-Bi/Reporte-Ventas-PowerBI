# Sales Dashboard in Power BI

This project presents an interactive sales report built with Power BI. It includes a detailed analysis of total sales, manufacturer performance, annual trends, and geographic distribution across different cities in Spain.

Key visualizations:
• Total Sales KPI displaying overall revenue.
• Sales by Manufacturer ranked by performance.
• Yearly Sales Analysis showing growth trends from 2016 to 2019.
• Sales by City using a geographic map for spatial insights.

This dashboard demonstrates data modeling, visualization design, and business analysis skills using Power BI.
